Setting A Static Page As Your Homepage

Default Home Page

  • Fresh installs see post previews
  • You can override this

Static vs Dynamic

  • Static pages remain roughly the same
  • Dynamic pages are those who reveal post previews

Blog Archive

  • You do not need any content within the page named "Blog". This can be an entirely blank page entitled "Blog".
  • Previews or excerpts of your blog posts will appear on this blank page as a post archive.

To set your preference:

  • Settings > Reading
  • Select A static home page
  • Choose your front page and posts page
  • Save your changes
